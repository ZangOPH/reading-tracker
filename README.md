# Reading Tracker

A personal reading tracker for web novels, manga, manhwa, and manhua — built with Python, Streamlit, FastAPI, and SQLite.

## Tech Stack

- **Frontend:** Streamlit
- **Backend:** FastAPI
- **Database:** SQLite
- **Data Sources:** MangaDex API, Anilist GraphQL, NovelUpdates, Royal Road

## Phases

### Phase 1 — Personal Tracker (Current)
- Log and track reading progress across all formats and sources
- Search and add titles manually or via API
- View reading history and stats

### Phase 2 — Recommendation Engine
- Cluster analysis on reading history
- Content-based filtering by genre, tags, and format
- Personalised recommendations

### Phase 3 — Community Product
- Multi-user support
- Shared lists and community recommendations
- Public-facing web app
